# Cafe Website Project ☕

## 📌 Description
This is a simple multi-page cafe website built using HTML and CSS.  
It includes different sections like Coffee, Drinks, Bakery, and Contact page.

## 📂 Project Structure
- index.html → Home page  
- coffe.html → Coffee menu  
- drinks.html → Drinks menu  
- bakery.html → Bakery items  
- contact.html → Contact page  
- style.css → Main styling  
- coffe.css → Coffee page styling  
- drinks.css → Drinks page styling  
- bakery.css → Bakery page styling  
- contact.css → Contact page styling  

## 🚀 Features
- Multi-page website
- Clean UI design
- Simple navigation bar
- Responsive basic layout

## 🛠️ Tools Used
- HTML5  
- CSS3  
- Git & GitHub
